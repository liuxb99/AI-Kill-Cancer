# v0.4.1 — Phase 2B Hardening Implementation Plan

## 当前状态
- EvidenceMerger 返回 dict，不持久化到数据库
- KnowledgeSourceModel / EvidenceItemModel / DrugInteractionModel 表已存在（migration 006）但从未写入
- POST /evidence/refresh 仅清空内存缓存，不查询源
- 无匹配优先级链（HGVS → coordinates → exact variant → molecular profile → gene fallback）
- 无 match_level 追踪
- 无 upsert/versioning/history
- 无冲突证据跟踪
- source-native level 与 normalized tier 未分离

## 缺失功能
1. 真正的持久化：EvidenceMerger 写入 KnowledgeSourceModel / EvidenceItemModel / DrugInteractionModel
2. 匹配优先级链
3. match_level 枚举
4. upsert + payload hash + first_seen_at / last_seen_at
5. withdrawn / superseded 状态跟踪
6. 冲突证据方向保存
7. source-native level 与 normalized tier 分开列
8. POST /evidence/refresh 真正刷新
9. POST /api/v1/evidence/cache/invalidate
