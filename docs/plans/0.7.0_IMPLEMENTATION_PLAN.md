# v0.7.0 — Clinical Reasoning Layer Implementation Plan

## Components
- ClinicalReasoningService — orchestrates reasoning
- ReasoningContextBuilder — builds frozen context snapshots
- EvidenceCitationValidator — validates citations
- HallucinationGuard — detects hallucinations
- ConflictAnalyzer — analyzes evidence conflicts
- ReasoningRunRepository — persists reasoning runs
