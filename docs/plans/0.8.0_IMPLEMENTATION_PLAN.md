# v0.8.0 — Clinical Report System Implementation Plan

## Components
- ReportBuilder — assembles report from snapshots
- ReportTemplateRegistry — manages HTML templates
- ReportValidator — validates report completeness
- ReportRepository — persists reports with versioning
- ReportRenderer — renders HTML
- FHIRExporter — FHIR-compatible export

## API
- POST /reports/case/{case_id}
- GET /reports/{report_id}
- GET /reports/{report_id}/html
- GET /reports/{report_id}/json
- GET /reports/{report_id}/fhir
