# 技術選型文件

## 前端
- **框架**: React 18 + TypeScript
- **UI**: Tailwind CSS + shadcn/ui
- **可視化**: D3.js / ECharts
- **建構**: Vite

## 後端
- **框架**: FastAPI (Python)
- **認證**: JWT + OAuth2
- **API 文檔**: Swagger (自動)

## AI/ML
- **框架**: PyTorch 2.x
- **NLP**: BioBERT / PubMedBERT
- **分子**: RDKit + PyTorch Geometric
- **實驗管理**: MLflow

## 資料庫
- **關聯**: PostgreSQL (結構化數據)
- **文檔**: MongoDB (非結構化/基因數據)
- **快取**: Redis

## 部署
- **容器**: Docker + Docker Compose
- **雲端**: AWS / GCP
- **監控**: Prometheus + Grafana
- **CI/CD**: GitHub Actions

## 開發工具
- **版本控制**: Git
- **語言**: Python 3.11+ / Node.js 20+
- **套件管理**: Poetry (Python) / npm (Node)
