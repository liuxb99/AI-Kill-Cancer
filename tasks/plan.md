# AI Kill Cancer 開發計劃

## 任務清單與優先級排序

---

### TASK-001：專案基礎架構建立
- **描述**：初始化專案目錄結構、設定開發環境（Python/Node.js）、建立版本控制（git）、配置 CI/CD 管線
- **依賴關係**：無
- **預估工時**：4 小時
- **負責角色**：exec-dev
- **優先級**：P0（最高）

---

### TASK-002：需求細化與技術選型
- **描述**：針對四大範疇（工具開發、網站建設、研究分析、開發整合）進行技術選型文件撰寫，確定各子系統的語言、框架、資料庫、部署方式
- **依賴關係**：TASK-001
- **預估工時**：3 小時
- **負責角色**：planner
- **優先級**：P0

---

### TASK-003：癌症診斷 AI 工具 - 資料收集與預處理
- **描述**：收集公開癌症數據集（TCGA、GEO、ICGC 等），編寫資料清洗與標準化管線，產出特徵工程 notebook
- **依賴關係**：TASK-002
- **預估工時**：8 小時
- **負責角色**：data-cleaner
- **優先級**：P1

---

### TASK-004：癌症診斷 AI 模型開發
- **描述**：建立深度學習/機器學習模型（分類、預後預測），支援多癌種（肺癌、乳腺癌、大腸癌等），輸出模型權重與評估報告
- **依賴關係**：TASK-003
- **預估工時**：16 小時
- **負責角色**：data-analyst
- **優先級**：P1

---

### TASK-005：治療推薦系統
- **描述**：基於患者基因組學與臨床數據，開發 AI 治療推薦引擎（用藥建議、療效預測），產出 API 模型
- **依賴關係**：TASK-004
- **預估工時**：12 小時
- **負責角色**：backend-logic
- **優先級**：P1

---

### TASK-006：後端 API 開發
- **描述**：開發 RESTful API（FastAPI/Flask），提供診斷預測、治療推薦、數據查詢等端點，整合 JWT 認證與 API 文件（Swagger）
- **依賴關係**：TASK-004, TASK-005
- **預估工時**：10 小時
- **負責角色**：api-designer
- **優先級**：P1

---

### TASK-007：資料庫設計與資料管線
- **描述**：設計 PostgreSQL/MongoDB 資料模型，建立 ETL 資料管線（定時更新癌症數據集、文獻摘要），配置資料備份策略
- **依賴關係**：TASK-003, TASK-006
- **預估工時**：8 小時
- **負責角色**：db-modeler
- **優先級**：P1

---

### TASK-008：資訊平台網站 - 前端開發（核心頁面）
- **描述**：建立 React/Vue 前端，包含首頁、癌症知識庫、工具入口、研究論文列表等核心頁面，實現響應式設計
- **依賴關係**：TASK-002
- **預估工時**：12 小時
- **負責角色**：ui-designer, frontend-logic
- **優先級**：P2

---

### TASK-009：可視化儀表板
- **描述**：開發互動式數據可視化儀表板（D3.js/ECharts），展示癌症統計數據、模型預測結果、研究趨勢圖表
- **依賴關係**：TASK-004, TASK-008
- **預估工時**：10 小時
- **負責角色**：data-vis
- **優先級**：P2

---

### TASK-010：文獻分析 AI 模型
- **描述**：開發 NLP 模型（基於 BioBERT/PubMedBERT）進行癌症文獻自動摘要、關鍵實體提取、研究趨勢分析
- **依賴關係**：TASK-003
- **預估工時**：10 小時
- **負責角色**：data-analyst
- **優先級**：P2

---

### TASK-011：藥物發現 AI 模型
- **描述**：建立分子生成與藥物-靶點相互作用預測模型，整合公開藥物數據庫（DrugBank、ChEMBL），產出候選藥物清單
- **依賴關係**：TASK-003
- **預估工時**：14 小時
- **負責角色**：data-analyst
- **優先級**：P2

---

### TASK-012：開源專案整合
- **描述**：整合現有開源癌症相關專案（如 DeepVariant、STAR 等），建立統一的 Docker 容器化部署方案
- **依賴關係**：TASK-006, TASK-007
- **預估工時**：6 小時
- **負責角色**：exec-dev
- **優先級**：P2

---

### TASK-013：知識庫與文件系統
- **描述**：建立專案知識庫（癌症百科、工具使用指南、API 文件、模型說明），配置自動化文檔生成
- **依賴關係**：TASK-002
- **預估工時**：6 小時
- **負責角色**：doc-writer
- **優先級**：P2

---

### TASK-014：研究入口網站 - 高階功能
- **描述**：開發研究人員入口（論文提交、數據上傳、模型測試沙箱、協作討論區）
- **依賴關係**：TASK-008, TASK-009
- **預估工時**：10 小時
- **負責角色**：frontend-logic, backend-logic
- **優先級**：P3

---

### TASK-015：測試與品質保證
- **描述**：為所有模組撰寫單元測試、整合測試、端到端測試，配置自動化測試執行，壓力測試 API
- **依賴關係**：TASK-006, TASK-008, TASK-009, TASK-010, TASK-011
- **預估工時**：10 小時
- **負責角色**：unit-tester, integration-tester
- **優先級**：P3

---

### TASK-016：系統部署與上線
- **描述**：配置雲端部署（AWS/GCP/Azure）、域名設置、SSL、監控（Prometheus/Grafana）、日誌系統
- **依賴關係**：TASK-012, TASK-015
- **預估工時**：8 小時
- **負責角色**：exec-dev
- **優先級**：P3

---

### TASK-017：使用者測試與反饋迭代
- **描述**：邀請早期使用者測試、收集反饋、迭代修復、撰寫使用者手冊
- **依賴關係**：TASK-016
- **預估工時**：8 小時
- **負責角色**：basic-tester, doc-writer
- **優先級**：P3

---
